# ProyectoFinal

# Funcionamiento del programa: 
Es una herramienta digital que automatiza el cálculo de seguros para TK-U, permitiendo procesar múltiples cotizaciones en una sola sesión mediante un bucle, reemplazando el lento proceso manual anterior.
# Partes del problema consideradas: 
El programa evalúa factores de riesgo individuales (edad del asegurado), familiares (edad del cónyuge y número de hijos) y económicos (propiedades y salario) para determinar los recargos, siempre calculados sobre un precio base de Q.2,000.
# Propuestas de mejora:
1.  Implementar una interfaz gráfica más amigable en lugar de ventanas de prompt() y alert().
2.  Añadir validaciones de datos más estrictas para evitar errores si el usuario ingresa letras en campos numéricos.
